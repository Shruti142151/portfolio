Overview

TaskTrackr is a lightweight web-based to-do list app built using HTML, CSS, and JavaScript. It helps users manage daily tasks effectively through a clean, minimal interface.

Key Features

  • Create, edit, and delete tasks
  • Mark tasks as completed
  • Responsive design for mobile and desktop
  • Simple UI focused on productivity

Installation

No installation required. Simply open the index.html file in a web browser or host it using GitHub Pages.

How to Use

  1. Type your task in the input box
  2. Click "Add" to insert it into the list
  3. Click the checkbox to mark it completed
  4. Use the trash icon to delete tasks

Frequently Asked Questions

Q1: Does it store my data permanently?

No, TaskTrackr does not use a backend or local storage. Refreshing the page resets tasks.

Q2: Can I use it on my phone?

Yes! It is fully responsive and works on mobile browsers.
